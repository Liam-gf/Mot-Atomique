# Mot Atomique

Mot Atomique est une petite application Java qui permet de vérifier si un mot peut être composé uniquement avec des symboles chimiques du tableau périodique des éléments.
Le programme affiche graphiquement les symboles correspondant à chaque lettre du mot s’il est valide.

## Lancer le projet

### Sur Windows

1. Double-cliquez sur `start.bat`
   **ou**
   Ouvrez PowerShell dans le dossier du projet.
2. Tapez :
   start.bat

### Linux

1. Ouvrez un terminal dans le dossier du projet.
2. Rendez le script exécutable :
   chmod +x start.sh

3. Executez le script :

./start.sh
